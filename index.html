Um pouco sobre HTML, CSS e Java Script

HTML

  • Hyper Text Markup Language (Linguagem de Marcação de Hipertexto).
  • Serve para introduzir o conteúdo da página web, tal como os textos e as mídias, adicionando toda a semântica dos mesmos. Dizendo o que é o cabeçalho, título, parágrafo, imagem, etc. Através das TAGS.Criando assim a estrutura do conteúdo da página
  • Não é uma linguagem de programação e sim uma linguagem de marcação, como ja diz o nome.
  • É errado dizer que se programa em HTML, o certo é dizer que se desenvolve em HTML
