TAGS HTML5 (Iniciante)

As TAGS que estiverem dentro de ( ), ao lado, são comandos que podem ser incrementados.

Semânticas

  • Títulos por hierarquia
    • <h1> <h2> <h3> <h4< <h5>
  • Parágrafo
    • <p>
  • Quebra de linha
    • <br>
  • Manter formatação igual à do código
    • <pre></pre>
  • Linha horizontal
    • <hr>
  • Termo em destaque (Negrito)
    • <strong></strong>
  • Termo com ênfase (Itálico)
    • <em></em>
  • Marca texto
    • <mark></mark>
  • Texto pequeno
    • <small></small>
  • Texto deletado (Riscado)
    • <del></del>
  • Texto inserido (Sublinhado)
    • <ins></ins>
  • Texto sobrescrito (32)
    • <sup></sup>
  • Subescrito (H2O)
    • <sub></sub>
  • Abreviações
    • <abbr title="Significado da Sigla"></abbr>
  • Citações (Usa-se <q></q> ao invés de digitar aspas para interpretadores de código entenderem a semântica)
    Ex: <p>Como diria o pai de um amigo: <q>O computador é um burro muito rápido</q></p>
    1. Parafrase curta
      • <q><p></p></q>
    2. Parafrase longa com link de referência
      • <blockquote cite="Link"><q></q></blockquoteq>
  • Comentários
    • <!-- Comentário Aqui -->
  • Símbolos especiais ⇽ Clique
    • Deve-se copiar o código que vem seguido de & e colar onde deseja que o símbolo apareça
  • Adicionar ícone à aba (<head>)
    • <link rel="shortcut icon" href="caminho/arquivo.ico" type="image/x-icon">

Listas

  • <ol></ol> Lista ordenada (type="1""A""a""I""i"")(start="1"""2""3")
  • <ul></ul> Lista desordenada (type="circle""disc""square")
  • <li> A lista em si

Hiperlinks

  • <a href="Link"></a> Link Externo (target="_blank" rel="external""nofollow")
  • <a href="Arquivo"></a> Link interno ("Caminho/Arquivo""../Arquivo" rel="next""prev")
  • <a href="Arquivo" download="Arquivo" type="aplication/Extensão"></a> Baixar arquivo

imagem

  • <picture></picture> indica que vai aparecer uma imagem

Imagem dinâmica (Tomando como exemplo 3 imagens de mesmo conteúdo, de 150px, 750px e 1000px respectivamente)

        <picture>
            <source media="(max-width: 750px)" srcset="Caminho/Arquivo150px" type="image/Extensão">
            <source media="(max-width: 1050px)" srcset="Caminho/Arquivo700px" type="image/Extensão">
            <img src="Caminho/Arquivo1000px" alt="Descrição Da Imagem">
        </picture>
*Sempre na ordem do maior para o menor, ou do menor para o maior (no caso usando min-width)

Algumas considerações:

Declarações iniciais de um código HTML:

No início de uma página html devemos usar sempre <!DOCTYPE html>, para indicar ao brownser que iremos trabalhar com HTML5, além de usar <html> logo após.

Usa-se, logo após a tag <head> a tag <meta charset="UTF-8">( que indica ao navegador que iremos trabalhar com o padrão de escrita UTF-8. O mais comum na www), além de outros elementos que também compõem a <meta>, como por exemplo <meta name="viewport" content="width=device-width, initial-scale=1.0">, para que o navegador identifique como a página deverá interpretar o códio da página

As declarações meta devem estar nos primeiros 1024 bytes da página, então quanto antes forem mensionadas, melhor.

caracteres especiais:

Seperadamente dos caracteres especiais, usados na sintaxe do HTML, Não é necessário uso de de referência de caracteres (& seguido do código). Pois os navegadores usam internamente o padrão unicode. O que significa que podem lidar com todos os caracteres definidos no padrão unicode.

Então a melhor prática é aplicar os símbolos diretamente no código.

Exemplo: <p>This is © 2015. Breach will entail a fine of € 400</p>

Não há necessidade de escrever & copy; ou & euro;
